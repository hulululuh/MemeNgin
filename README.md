## Building

```
git clone https://github.com/njbrown/fxmixer.git

cd fxmixer

# if you want to pull down assets (textures and node icons)
git submodule update --init

```

yarn:

```
yarn install
yarn electron:serve
```

npm:

```
npm install
npm run electron:serve
```
