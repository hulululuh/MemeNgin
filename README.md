## Building

```
git clone git@github.com:hulululuh/MemeNgin.git

cd MemeNgin

# if you want to pull down assets (textures and node icons)
git submodule update --init

```

yarn:

```
yarn install
yarn electron:serve
```

npm:

```
npm install
npm run electron:serve
```
